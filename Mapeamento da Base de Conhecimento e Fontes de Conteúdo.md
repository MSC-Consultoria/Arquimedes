# Mapeamento da Base de Conhecimento e Fontes de Conteúdo

## Introdução

A construção de uma base de conhecimento robusta e diversificada é fundamental para o sucesso do sistema educacional personalizado. Este documento organiza as fontes de conteúdo identificadas por domínio de conhecimento, priorizando materiais gratuitos, de alta qualidade e com profundidade adequada para atender desde iniciantes até usuários avançados.

## Estrutura de Organização do Conteúdo

Cada fonte será categorizada de acordo com os seguintes critérios:

- **Domínio de Conhecimento**: Área principal do conteúdo (ex: Tecnologia, Concursos, Disciplinas Fundamentais).
- **Tópico**: Tema específico dentro do domínio (ex: Java, OAB, Matemática).
- **Nível de Dificuldade**: Iniciante, Intermediário ou Avançado.
- **Tipo de Conteúdo**: Vídeo (YouTube), PDF, Texto, Interativo.
- **Qualidade**: Avaliação baseada em reputação, profundidade e didática.

## 6. Estratégia de Expansão da Base de Dados

### Fase 1: Coleta Inicial (MVP)
Priorizar as fontes de maior qualidade e abrangência para criar uma base inicial funcional:
- **Curso em Vídeo** (Tecnologia)
- **Khan Academy** (Matemática)
- **Gran Cursos OAB** e **Estratégia OAB** (Concursos)
- **Livros da Caelum** (Java)

### Fase 2: Expansão Progressiva
Adicionar fontes complementares e aprofundar subtópicos:
- Canais internacionais de programação (Net Ninja, Bro Code)
- Materiais de concursos específicos (DETRAN, outros)
- Livros avançados de infraestrutura e cloud

### Fase 3: Curadoria Contínua
Manter um processo contínuo de:
- Avaliação da qualidade do conteúdo
- Atualização com novos materiais
- Remoção de fontes obsoletas ou de baixa qualidade
- Feedback dos usuários sobre a utilidade do conteúdo

---

## 7. Processo de Extração e Estruturação

Para cada fonte identificada, seguiremos o seguinte fluxo:

1. **Identificação**: Catalogar a fonte com todos os metadados (URL, autor, data, tópico).
2. **Extração**: Utilizar scripts automatizados para extrair o conteúdo (transcrições de vídeos, texto de PDFs).
3. **Curadoria**: Revisão humana para garantir qualidade e corrigir erros.
4. **Estruturação**: Formatação em JSON seguindo o modelo de dados definido.
5. **Armazenamento**: Inserção no banco de dados com tags e referências.
6. **Indexação**: Preparação para busca e recomendação pelo motor de personalização.

---

## 8. Referências e Profundidade

Conforme especificado nos requisitos, cada conteúdo terá **referências de referências**, criando uma rede de conhecimento profunda:

- **Referências Primárias**: Fonte original do conteúdo (ex: vídeo do YouTube, PDF).
- **Referências Secundárias**: Materiais citados ou recomendados pela fonte primária (ex: documentação oficial, livros).
- **Referências Terciárias**: Fontes adicionais para aprofundamento (ex: artigos acadêmicos, tutoriais avançados).

Esta abordagem garante que o aluno tenha acesso não apenas ao conteúdo principal, mas também a um caminho claro para se aprofundar em qualquer tópico de interesse.
