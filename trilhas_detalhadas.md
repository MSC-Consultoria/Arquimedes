## 1. Inglês - Leitura e Desenvolvimento Contínuo

**Objetivo**: Desenvolver habilidades de leitura e compreensão em inglês, com foco em textos técnicos de programação e tecnologia.

### Módulo 1: Fundamentos do Inglês
**Subtópicos:**
- Alfabeto e pronúncia básica (sons das vogais e consoantes)
- Números, dias da semana e meses
- Cumprimentos e expressões cotidianas
- Estrutura básica de frases (sujeito + verbo + complemento)

**Analogia**: Aprender inglês é como aprender a tocar um instrumento - você começa com notas básicas antes de tocar músicas completas.

### Módulo 2: Vocabulário Essencial (500 Palavras)
**Subtópicos:**
- 100 palavras mais comuns (the, a, is, are, have, etc.)
- Verbos de ação essenciais (go, make, do, get, see)
- Substantivos do dia a dia (house, car, food, water)
- Adjetivos descritivos (big, small, good, bad)
- Vocabulário técnico básico (computer, program, code, data)

### Módulo 3: Gramática Fundamental
**Subtópicos:**
- Tempos verbais: Present Simple e Present Continuous
- Artigos: a, an, the
- Pronomes pessoais e possessivos
- Preposições de lugar e tempo (in, on, at)
- Formação de perguntas (What, Where, When, Who, Why, How)

### Módulo 4: Leitura de Textos Simples
**Subtópicos:**
- Estratégias de leitura (skimming e scanning)
- Identificação de palavras-chave
- Compreensão de contexto
- Prática com textos curtos (notícias, emails, posts)

### Módulo 5: Inglês Técnico para Programação
**Subtópicos:**
- Vocabulário de programação (variable, function, loop, array)
- Leitura de documentação técnica
- Compreensão de mensagens de erro
- Leitura de tutoriais e artigos técnicos
- Participação em fóruns (Stack Overflow, Reddit)

## 2. N8N - Automação de Workflows

**Objetivo**: Dominar a ferramenta N8N para criar automações que economizam tempo e aumentam produtividade.

### Módulo 1: Introdução à Automação
**Subtópicos:**
- O que é automação? (analogia com máquinas que fazem tarefas repetitivas)
- Exemplos de automação no dia a dia (alarme do celular, cafeteira programável)
- Benefícios da automação no trabalho
- Quando automatizar vs fazer manualmente

**Caso Real**: Automatizar o envio de relatórios semanais por email economiza 2 horas por semana.

### Módulo 2: Conhecendo o N8N
**Subtópicos:**
- O que é N8N e para que serve
- Instalação local vs versão cloud
- Interface do N8N (canvas, nodes, connections)
- Conceitos básicos: workflows, nodes, triggers, actions

### Módulo 3: Primeiro Workflow Simples
**Subtópicos:**
- Criando um workflow do zero
- Adicionando um trigger (webhook, schedule)
- Conectando nodes
- Testando e debugando
- Ativando o workflow

**Projeto Prático**: Criar workflow que envia email toda segunda-feira às 9h.

### Módulo 4: Integrações Essenciais
**Subtópicos:**
- Google Sheets (ler e escrever dados)
- Gmail (enviar emails automaticamente)
- Webhooks (receber dados de outros sistemas)
- HTTP Request (fazer chamadas a APIs)
- Slack/Discord (enviar notificações)

### Módulo 5: Workflows Avançados
**Subtópicos:**
- Condicionais (IF nodes)
- Loops e iterações
- Manipulação de dados (Set, Function nodes)
- Tratamento de erros
- Workflows complexos com múltiplas integrações

**Projeto Final**: Sistema de monitoramento que verifica preços de produtos e envia alerta quando houver desconto.

## 3. APIs e Integração

**Objetivo**: Compreender o funcionamento de APIs e saber consumir e criar integrações.

### Módulo 1: Fundamentos de APIs
**Subtópicos:**
- O que é uma API? (analogia com cardápio de restaurante)
- Cliente e servidor (quem pede e quem atende)
- Protocolo HTTP (a "linguagem" da web)
- Endpoints (os "endereços" das funcionalidades)

**Analogia**: Uma API é como um garçom em um restaurante - você faz o pedido (request) e ele traz o que você pediu (response).

### Módulo 2: Métodos HTTP
**Subtópicos:**
- GET (buscar dados - como pedir para ver o cardápio)
- POST (criar dados - como fazer um pedido)
- PUT (atualizar dados - como modificar um pedido)
- DELETE (remover dados - como cancelar um pedido)
- PATCH (atualizar parcialmente)

### Módulo 3: JSON - Formato de Dados
**Subtópicos:**
- O que é JSON (JavaScript Object Notation)
- Estrutura de objetos e arrays
- Chaves e valores
- Leitura e escrita de JSON
- Ferramentas para visualizar JSON (JSON Viewer, Postman)

### Módulo 4: Autenticação em APIs
**Subtópicos:**
- API Keys (chaves de acesso)
- Bearer Tokens
- OAuth 2.0 (login com Google, Facebook)
- Headers de autenticação
- Segurança e boas práticas

### Módulo 5: Consumindo APIs Reais
**Subtópicos:**
- Usando Postman para testar APIs
- Consumindo APIs com JavaScript (fetch, axios)
- Consumindo APIs com Python (requests)
- Tratamento de erros (status codes: 200, 404, 500)
- Rate limiting e quotas

**Projetos Práticos**:
- Buscar dados de clima (OpenWeather API)
- Buscar informações de CEP (ViaCEP)
- Integrar com API do GitHub

## 4. GitHub - Versionamento e Colaboração

**Objetivo**: Dominar Git e GitHub para versionar código e colaborar em projetos.

### Módulo 1: Conceitos de Controle de Versão
**Subtópicos:**
- O que é controle de versão? (analogia com "Ctrl+Z" super poderoso)
- Por que versionar código?
- Histórico de mudanças
- Trabalho em equipe sem conflitos

**Analogia**: Git é como uma máquina do tempo para seu código - você pode voltar a qualquer versão anterior.

### Módulo 2: Instalação e Configuração
**Subtópicos:**
- Instalando Git no Windows/Mac/Linux
- Configuração inicial (nome, email)
- Criando conta no GitHub
- SSH keys vs HTTPS
- Interface do GitHub (repositórios, issues, pull requests)

### Módulo 3: Comandos Básicos do Git
**Subtópicos:**
- `git init` (iniciar repositório)
- `git clone` (copiar repositório)
- `git add` (preparar mudanças)
- `git commit` (salvar mudanças)
- `git push` (enviar para GitHub)
- `git pull` (baixar atualizações)
- `git status` (ver estado atual)

### Módulo 4: Branches e Merge
**Subtópicos:**
- O que são branches? (linhas paralelas de desenvolvimento)
- Criando e mudando de branch (`git branch`, `git checkout`)
- Merge (juntar branches)
- Resolvendo conflitos
- Estratégias de branching (main, develop, feature)

### Módulo 5: Colaboração e Open Source
**Subtópicos:**
- Forks (copiar repositório de outros)
- Pull Requests (propor mudanças)
- Code Review (revisar código)
- Issues (reportar bugs e sugerir features)
- Contribuindo para projetos open source

**Projeto Final**: Contribuir com uma correção ou melhoria em um projeto open source real.

## 6. CLI - Command Line Interface

**Objetivo**: Dominar a linha de comando para aumentar produtividade e automatizar tarefas.

### Módulo 1: Por Que Usar a Linha de Comando?
**Subtópicos:**
- Vantagens sobre interface gráfica
- Velocidade e eficiência
- Automação de tarefas repetitivas
- Acesso a ferramentas avançadas

**Analogia**: CLI é como dirigir com câmbio manual - mais controle e poder, mas requer prática.

### Módulo 2: Navegação em Diretórios
**Subtópicos:**
- `pwd` (onde estou?)
- `ls` (o que tem aqui?)
- `cd` (mudar de pasta)
- Caminhos absolutos vs relativos
- Atalhos: `~` (home), `.` (atual), `..` (pasta anterior)

### Módulo 3: Manipulação de Arquivos
**Subtópicos:**
- `mkdir` (criar pasta)
- `touch` (criar arquivo)
- `cp` (copiar)
- `mv` (mover/renomear)
- `rm` (deletar - cuidado!)
- `cat` (ver conteúdo de arquivo)

### Módulo 4: Pipes e Redirecionamento
**Subtópicos:**
- `|` (pipe - conectar comandos)
- `>` (redirecionar saída para arquivo)
- `>>` (adicionar ao final do arquivo)
- `<` (ler entrada de arquivo)
- `grep` (buscar texto)
- `find` (buscar arquivos)

### Módulo 5: Scripts Bash Básicos
**Subtópicos:**
- Criando primeiro script (.sh)
- Variáveis
- Estruturas de controle (if, for, while)
- Funções
- Automatizando tarefas do dia a dia

**Projeto Final**: Script que faz backup automático de arquivos importantes.

## 8. Fundamentos de Programação

**Objetivo**: Aprender lógica de programação e conceitos fundamentais, independente de linguagem.

### Módulo 1: Lógica de Programação (Sem Código)
**Subtópicos:**
- O que é um algoritmo? (receita de bolo)
- Sequência de passos
- Tomada de decisões (se isso, então aquilo)
- Repetição (fazer algo várias vezes)
- Pensamento computacional

**Analogia**: Programar é como dar instruções para alguém que segue tudo ao pé da letra.

### Módulo 2: Variáveis e Tipos de Dados
**Subtópicos:**
- O que são variáveis? (caixas com etiquetas)
- Números (inteiros e decimais)
- Textos (strings)
- Booleanos (verdadeiro/falso)

- Objetos/Dicionários (dados estruturados)

### Módulo 3: Estruturas de Controle
**Subtópicos:**
- Condicionais (if, else, elif)
- Operadores lógicos (and, or, not)
- Operadores de comparação (==, !=, >, <)
- Switch/Case
- Aninhamento de condicionais

### Módulo 4: Loops e Repetição
**Subtópicos:**
- Loop `for` (repetir N vezes)
- Loop `while` (repetir enquanto condição for verdadeira)
- `break` (sair do loop)
- `continue` (pular iteração)
- Loops aninhados

### Módulo 5: Funções e Modularização
**Subtópicos:**
- O que são funções? (máquinas que fazem tarefas específicas)
- Parâmetros e argumentos
- Retorno de valores
- Escopo de variáveis
- Reutilização de código

**Projeto Final**: Criar calculadora completa com todas as operações básicas.

## Metodologia de Ensino

Todas as trilhas seguem os mesmos princípios pedagógicos:

**Progressão Gradual**: Cada módulo introduz apenas 2-3 conceitos novos, consolidando bem antes de avançar.

**Analogias Constantes**: Cada conceito abstrato é explicado com analogia do mundo real.

**Exemplos Práticos**: Mínimo de 3 exemplos para cada conceito, com casos reais de aplicação.

**Projetos Hands-On**: Cada módulo termina com projeto prático para aplicar o conhecimento.

**Quizzes Frequentes**: Validação de compreensão a cada 10-15 minutos de conteúdo.

**Glossário Integrado**: Termos técnicos sempre explicados em linguagem simples.

**Suporte Visual**: Diagramas, infográficos e vídeos para facilitar compreensão.
